## **环境说明与运行指南**

### **一、环境要求**

#### **1. 开发环境**
- **操作系统**：Windows 11
- **IDE**：Android Studio
- **Android SDK**：Android 14 (API 34)

#### **2. 构建工具**
- **Gradle**：8.13
- **Kotlin**：2.0.21

#### **3. 依赖库版本**
| 库名称                               | 版本   | 说明                 |
| ------------------------------------ | ------ | -------------------- |
| androidx.core:core-ktx               | 1.17.0 | Kotlin 扩展库        |
| androidx.appcompat:appcompat         | 1.6.1  | 兼容库               |
| com.google.android.material:material | 1.10.0 | Material Design 组件 |
| androidx.recyclerview:recyclerview   | 1.3.2  | RecyclerView 列表    |
| androidx.swiperefreshlayout          | 1.1.0  | 下拉刷新             |
| androidx.constraintlayout            | 2.1.4  | 约束布局             |

---

### **二、项目运行步骤**

#### **使用 Android Studio**

1. 打开项目
   ```
   1. 启动 Android Studio
   2. 打开项目文件
   3. 等待 Gradle 同步完成
   ```
   
2. 等待 Gradle 同步
   
3. 连接设备或启动模拟器
   
   **选项 A：使用物理设备**
   ```
   1. 启用开发者选项和 USB 调试
   2. 用 USB 连接手机到电脑
   3. 在 Android Studio 中点击设备选择器
   4. 选择你的设备
   ```
   
   **选项 B：使用 Android 模拟器**
   ```
   1. 打开 Device Manager
   2. 点击 "Create Device"
   3. 选择设备型号
   4. 选择系统镜像
   5. 点击 "Finish" 创建
   6. 点击运行按钮启动模拟器
   ```

4. 运行项目
   ```
   1. 点击工具栏的绿色运行按钮
   ```
   
5. 等待构建和安装
   - 首次运行会编译项目（可能需要几分钟）
   - 构建完成后自动安装到设备
   - 应用自动启动
